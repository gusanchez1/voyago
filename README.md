# ✈️ Voyago — Website de Agência de Viagens

Voyago é um projeto acadêmico no qual fomos incentivados a desenvolver um site utilizando ferramentas modernas de desenvolvimento web.

## 🚀 Funcionalidades

* Estrutura de CSS modular
* Componentes reutilizáveis (botões, cards, navbar)
* Seção hero com imagem em tela cheia
* Múltiplas páginas (Início, Destinos, Contato)

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3 (arquitetura modular com variáveis)
* Bootstrap 5


## 🎯 Objetivos do Projeto

* Aprimorar habilidades em desenvolvimento front-end
* Praticar organização de CSS de forma limpa e escalável
* Desenvolver um site com aparência próxima a um projeto real
